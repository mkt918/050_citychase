# シティチェイス (City Chase)

警察vs犯人の非対称対戦ボードゲームのWebアプリ版

## 🎮 ゲーム概要

**シティチェイス**は、犯人1人と警察チーム(ヘリコプター3機)に分かれて対戦する推理ゲームです。

### ルール

#### 犯人側

- 5x5のグリッド上を移動
- 移動するたびに痕跡チップを残す
- 既に痕跡を置いた場所には戻れない
- 11ラウンド逃げ切れば勝利

#### 警察側

- ヘリコプター3機を操作
- 各ヘリは「移動」または「調査」のどちらか1つを実行
- ビルを調査して痕跡または犯人の車を発見
- 犯人の車を発見するか、犯人を包囲すれば勝利
